## ðŸš€ Quick Start

**MySQL is required for the bot and website to function.**

### Prerequisites
- Node.js v18+
- MySQL database
- Discord bot token

### Installation
```bash
git clone <repo-url>
cd oa1u-Bot
npm install
```

### Setup
1. Configure your MySQL database and fill in everything in the Config folder.
2. Create your first (owner) account:
   ```bash
   node scripts/createAccount.js
   ```
   Follow the prompts.
3. Start the bot:
   ```bash
   npm start
   ```
